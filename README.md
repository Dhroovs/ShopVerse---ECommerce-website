# 🛍️ NovaCart - Laravel E-Commerce Platform

NovaCart is a full-featured e-commerce web application built using Laravel 10. It includes authentication, product management, cart functionality, checkout system, and an admin panel — all wrapped in a modern, responsive UI.

---

## 🚀 Features

### 🔐 Authentication
- User registration & login
- Session-based authentication
- Secure password hashing

### 🛒 Shopping Experience
- Dynamic product listing
- Search functionality
- Product detail page
- Add to cart system
- Update/remove items
- Checkout process

### 💳 Orders
- Order placement
- Order tracking (basic)
- Order history page

### 🧑‍💼 Admin Panel
- Dashboard overview
- Product management
- Order management

### 🎨 UI/UX
- Modern responsive design
- Hero carousel & banners
- Product cards with hover effects
- Wishlist UI
- Toast notifications

---

## 🛠️ Tech Stack

- Laravel 10
- PHP 8+
- MySQL (XAMPP)
- Blade Templates
- Bootstrap 5
- JavaScript / jQuery
